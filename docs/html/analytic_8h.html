Detailed Description

Header file for setting library variables.

Author
Marco Mihai Condrache (marco.nosp@m.miha.nosp@m.icond.nosp@m.rach.nosp@m.e@gma.nosp@m.il.c.nosp@m.om)
Version
1.0
Date
25-03-2020

This API supports the following features:

  • setting divisor
  • setting scale
  • setting window size
  • setting terminal output
  • init the data set
  • set the data set

These functions must be called from the main thread.
