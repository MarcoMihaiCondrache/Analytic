)

Function to handle a menu.

This function shows the available options on the screen and
through a call Ui_readInput puts the selection on hold. If
the selected element does not exist, the function re-proposes
the menu and this is where the clear parameter acts. If true,
the function deletes the old text of the console and re-proposes
it following the wrong selection

Parameters
leadingMsg Text to print before the menu
selectionMsg Text to print before the input request (after the menu)
items Items of the menu
itemsCount The number of the items
clear Bool that defines if the menu has to clear every time the console
Returns
int The selected item
