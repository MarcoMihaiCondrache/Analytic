◆ Analytic_Hyperbole

typedef struct Analytic_Hyperbole Analytic_Hyperbole

Hyperbole structure.

Hyperbole types supported:

  • \(\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1\)
  • \(\frac{x^2}{a^2} - \frac{y^2}{b^2} = -1\)
  • \(xy = k\)
  • \(y = \frac{ax + b}{cx + d}\)
