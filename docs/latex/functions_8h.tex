Mathematical function to calculate the radius of a circumference. 

The function calculates the radius with a mathematic formula

$r = \sqrt{\frac{a^{2}}{4} + \frac{b^{2}}{4} - c}$

\begin{DoxyWarning}{Warning}
The function also checks for a negative radius. ~\newline
In this case the program will exit returning a fatal error. ~\newline
Before calling this function, check that the ~\newline
given circumference is really a circumference!
\end{DoxyWarning}

Mathematical function to get y of hyperboles with horizontal symmetry axis. 

The function calculates the y with a mathematic formula

$y = \pm \sqrt{b^{2} + \frac{a^{2}}{b^{2}}x^{2}}$

The function calculates the second y and the first is drawn by exploiting the symmetry property of the hyperbole for more precise coordinates

$y = - \sqrt{b^{2} + \frac{a^{2}}{b^{2}}x^{2}}$

The function also checks that the hyperbole parameters ~\newline
are larger and different from zero in order to avoid errors ~\newline
during mathematical calculations. If this were the case, ~\newline
the function would block the program and return a fatal error
